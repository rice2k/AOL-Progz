# Archive Safety

AOL Progz exists for preservation, research, and historical context.

The files in this archive are old ZIP/RAR/SIT/HQX packages from public archival
sources. Some may contain broken executables, prankware, abusive AOL-era tools,
password stealers, macro tools, mailers, punters, or other unsafe programs.

Recommended handling:

- Do not run archive contents on a real machine.
- Inspect files only inside an isolated virtual machine or emulator.
- Keep the VM disconnected from personal accounts and private networks.
- Treat every executable, script, DLL, OCX, and Visual Basic runtime component as
  untrusted.
- Preserve source links and filenames when researching provenance.

This site intentionally avoids operational instructions for using abusive tools.
