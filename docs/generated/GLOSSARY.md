# Glossary

Terms are preserved as AOL/AIM scene vocabulary. Abuse-related terms are documented only to explain old archive labels.

| Term | Type | Description |
| --- | --- | --- |
| Proggie / Prog / Progz | general | Independent AOL-era helper apps, usually made by hobbyists in Visual Basic. They ranged from chat and macro tools to all-in-one suites, bots, mailers, punters, faders, and account utilities. |
| Punter | abuse category | A tool associated with forcing another user offline or making their AOL session unstable. This site treats punters as historical artifacts and does not document operating steps. |
| Booter | abuse category | Scene synonym for a punter-style tool used to kick or boot someone offline. Preserved here only as historical vocabulary. |
| Anti / Anti Booter | defense tool | A program advertised as protecting a user from being kicked offline by punters or booters. These were part of the back-and-forth tooling culture around AOL chat rooms. |
| Room buster | abuse category | A tool or feature associated with repeatedly trying to enter a full room until an opening appears; the phrase was also used around disruptive room behavior in some archives. |
| Buster | room utility | Short scene label for a room buster or related room-entry utility. |
| C-Com | command/chat tool | A command-list tool where a typed chat trigger could send a stored list of commands or canned chat actions. |
| Scroller | chat tool | A program that automated large text, ASCII art, or repeated chat lines. Some scrollers tried to avoid room anti-scroll limits by pacing messages. |
| Macro studio | creative/chat tool | A tool for building, storing, and sending macro art or scripted chat text, often used for ASCII art, banners, and stylized room posting. |
| Fader | text effect | A program that generated color-faded or stylized text for chat, profiles, or messages. |
| Idler / AFK bot | automation | A bot that kept a screen name present, auto-responded, or performed simple unattended actions while the user was away. |
| MMer / Mass mailer | abuse category | AOL-era bulk mail or mail-bomb tooling. This archive records it as part of the scene history without reproducing usage instructions. |
| Server | file/mail sharing context | In this scene vocabulary, a server often meant a tool that announced or served a list of files, emails, or downloads in chat. Some overlapped with mass-mailer behavior. |
| Cracker | account abuse category | A label for tools associated with trying to obtain or break into account passwords. This was illegal and is included only for historical classification. |
| Phisher | account abuse category | A credential-theft or fake-login category used to trick someone into giving up an account password. This was illegal and is included only for historical classification. |
| X'er | text or chat utility | A scene label often used for tools that ignored or blocked annoying chat-room users, transformed text, generated effects, or performed a narrow AOL automation trick. |
| TOS / Termer | account abuse context | Tools and lore related to AOL Terms of Service reports, warnings, account termination, or attempts to manipulate those systems. |
| Termer | account abuse category | A tool category associated with trying to terminate or kill an AOL account. This was illegal and is included only as archive vocabulary. |
| OH account | account context | Scene shorthand often used around account status, billing, or overhead-account lore. Kept here as vocabulary, not as operational guidance. |
| Misc prog | mixed category | A catch-all label for programs that did not fit a single category or bundled many features into one tool. |
