# AOL/AIM Prog Era Timeline

This timeline gives the GitHub archive historical context for AOL, AIM, ICQ, MSN Messenger, old source sites, and later rescue work. It is context for browsing, not a compatibility guarantee for any single file.

| Year | Event | Context | Source |
| --- | --- | --- | --- |
| 1995 | AOL 2.5 era | AOL 2.5 became one of the important early Windows targets for classic progs such as AOHell-style tools. | [https://www.oldversion.com/software/america-online/](https://www.oldversion.com/software/america-online/) |
| 1996 | AOL 3.0 | AOL 3.0 broadened the Windows audience for proggies, punters, macro tools, and chat automation. | [https://www.oldversion.com/software/america-online/](https://www.oldversion.com/software/america-online/) |
| 1997 | AIM breaks out | AOL Instant Messenger helped separate instant messaging from the full AOL client and became its own target for tools and experiments. | [https://en.wikipedia.org/wiki/AIM_(software)](https://en.wikipedia.org/wiki/AIM_(software)) |
| 1998 | AOL 4.0 and ICQ | AOL 4.0 arrived, and AOL also acquired Mirabilis, the company behind ICQ, as instant messaging became central internet culture. | [https://www.wired.com/1998/06/aol-grabs-icq-firm/](https://www.wired.com/1998/06/aol-grabs-icq-firm/) |
| 1999 | AOL 5.0 and MSN Messenger | AOL 5.0 and MSN Messenger arrived in the same broader era, expanding the chat ecosystem beyond AOL/AIM. | [https://news.microsoft.com/source/1999/07/21/microsoft-launches-msn-messenger-service/](https://news.microsoft.com/source/1999/07/21/microsoft-launches-msn-messenger-service/) |
| 2000-2003 | AOL 6.0 through 9.0 | Later AOL clients changed the target surface while older proggies, source snippets, and Visual Basic components continued to circulate. | [https://www.oldversion.com/software/america-online/](https://www.oldversion.com/software/america-online/) |
| 2016-2026 | Rescue and catalog work | GitHub and Wayback-based preservation projects began turning scattered ZIPs, screenshots, readmes, and old domains into searchable archives. | [https://github.com/ssstonebraker/aolunderground-proggies](https://github.com/ssstonebraker/aolunderground-proggies) |
