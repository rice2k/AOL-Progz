Files On The Site

Each mirrored catalog item uses a cleaned local path under files/, so downloads stay attached to this GitHub Pages site. Original filenames and upstream GitHub source paths remain visible on every card.

Very large off-site archives, such as the Kadeklizem RAR, remain recorded as Wayback mirrors because ordinary GitHub repositories reject oversized single files.
